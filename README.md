1. Nesprávne rozdelenie úloh medzi tímom

1. Výpadok servera pri zvýšenom zaťažení

1. Nezohľadnenie bezpečnostných aspektov
1. Pretečenie zásobníka (stack overflow)
1. Nezabezpečené uchovávanie hesiel
